# Diagrams

## Purpose

Store source diagrams and exported images that explain the platform visually.

## Folder Structure

| Path | Purpose |
|---|---|
| `diagrams/architecture/` | Platform, service, and dependency diagrams |
| `diagrams/networking/` | Logical topology, VLAN, and IP maps |
| `diagrams/rack/` | Rack elevation and cabling visuals |
| `diagrams/power/` | UPS, PDU, and power distribution diagrams |
| `diagrams/recovery/` | Recovery order, backup flow, and DR visuals |
| `diagrams/ot-lab/` | OT/ICS topology and scenario diagrams |

## Standards

- Keep editable source files such as `.drawio`, `.vsdx`, or `.svg` alongside exported `.png` files.
- Match diagram names to the document that references them.
- Redact anything sensitive before committing exported images.
