# Assets

## Purpose

Store reusable repository assets such as documentation templates, reference material, and media used across the project.

## Folder Structure

| Path | Purpose |
|---|---|
| `assets/templates/` | Reusable document and table templates |
| `assets/reference/` | Public-safe reference material and supporting notes |
| `assets/media/` | Shared media used by docs or portfolio exports |

## Standards

- Keep reusable assets generic and sanitized.
- Prefer assets that improve documentation consistency across future phases.
- Link templates from relevant docs when they become canonical.
